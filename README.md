# Expo Starter

A production-ready Expo starter template with essential tools for mobile app development and distribution.

## Features

- **Expo App** - Created with `npx create-expo-app@latest`
- **GitHub Pages Website** - Includes support, privacy policy, and roadmap pages hosted via GitHub Pages

## Getting Started

```bash
cd expo-starter
npm install
npm start
```

## Website

The `docs/` directory contains markdown files for the GitHub Pages site, providing essential pages for your app.
